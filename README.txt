Authors: Julian Gardner, Doug Williams

Much of this code is not mine but was written by Doug Williams, one of the professors at my college.
I have marked which lines were written by me.
NO PULL REQUESTS PLEASE - this is kept only for archival purposes.

INSTALLATION
1. Download and install Racket if you don't already have it installed.
2. Install the csv-reading package.
3. Put the csv file and the jpg file in the same directory as the program file ending in .rkt
4. Run meteorites.rkt using your Racket interpreter.
